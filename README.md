# CrudMobile

## Description 🎮
trabalho CrudMobile para a materia de gerencia e configuração de softwares da universidade unisales de vitoria.

## Technologies 💻
* Vue
* Capacitor Android
* Dexie
* html
* css
* javascript

## Pre-requisitos
* npm v 9
* node v 18

## Installation 🛠️

```bash
$ npm install
```

## Running the API ⚡

```bash
$ npm run dev
```
